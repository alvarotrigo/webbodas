Quick Setup

Create Clerk Account

Sign up at clerk.com and create a new application. Choose JavaScript as your framework.

Configure Authentication

Enable Google, GitHub, and Magic Link authentication in your Clerk Dashboard under User & Authentication.

Update Configuration

Edit clerk-config.js and replace the placeholder values with your actual Clerk credentials:

const CLERK_CONFIG = { publishableKey: 'pk_test_YOUR_ACTUAL_KEY', frontendApiUrl: 'https://YOUR_ACTUAL_URL', redirectUrls: ['http://localhost:5173'] };

Test Authentication

Start your development server and navigate to auth-wall.html to test all authentication methods.
